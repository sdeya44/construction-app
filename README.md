# ניהול אתרי בנייה — Production v2

מערכת PWA לניהול יומי של אתרי בנייה. מבוססת על Google Sheets ו-Google Drive.

## קבצים במערכת

```
index.html          — מסך הבית (HTML בלבד, ללא קוד)
manifest.json       — הגדרות PWA
sw.js               — Service Worker (קאש + עדכונים)
css/
  app.css           — עיצוב כל המסכים
js/
  config.js         — מזהים, כותרות טבלאות, הרשאות
  utils.js          — פונקציות עזר (תאריך, דחיסה, CSV)
  api.js            — גישה ל-Google Sheets ו-Drive
  state.js          — נתוני האפליקציה (D) וטעינה
  auth.js           — התחברות Google
  screens/
    dashboard.js    — דשבורד
    logs.js         — יומנים וחיפוש
    employees.js    — עובדים
    sites.js        — אתרים ותמונות
    management.js   — ספקים וציוד
    wizard.js       — אשף דיווח יומי
    photos.js       — העלאת תמונות
    reports.js      — דוחות PDF/CSV ונעילת חודשים
```

## גיליונות Google Sheets נדרשים

המערכת יוצרת את הגיליונות אוטומטית בעת הכניסה הראשונה.

| גיליון | תיאור |
|--------|--------|
| Sites | אתרי הבנייה |
| Employees | עובדים |
| Equipment | ציוד |
| Suppliers | ספקים |
| DailyLogs | יומנים יומיים |
| Attendance | נוכחות עובדים |
| LogEquipment | ציוד שבוצע ביומן |
| Deliveries | משלוחים וחומרים |
| MonthLocks | נעילת חודשים |
| SitePhotos | תמונות אתרים |
| AuditLog | לוג פעולות (יצירה/עדכון/מחיקה) |
| Users | משתמשים ותפקידים |

## תפקידים

| תפקיד | הרשאות |
|--------|--------|
| Admin | הכל |
| Manager | צפייה, יצירה, עדכון (ללא נעילה ומחיקה) |
| Viewer | צפייה בדוחות בלבד |

כדי להגדיר תפקיד: הוסף שורה לגיליון **Users** עם עמודות:
`UserID, Email, Name, Role, AddedAt, AddedBy`

משתמש שאינו ברשימה יקבל תפקיד Admin אוטומטית.

## הגנת גרסאות (Version Control)

כל יומן יומי כולל שדה **Version**. אם שני אנשים ינסו לערוך יומן באותו זמן, האחד יקבל הודעת שגיאה ויצטרך לטעון מחדש לפני שמירה.

## גיבוי אוטומטי

כל פעולה (יצירה, עדכון, מחיקה, נעילה) נרשמת בגיליון **AuditLog**.

## רענון אוטומטי

המערכת מרעננת נתונים כל 45 שניות (לא בזמן מילוי דיווח).

## פריסה ב-GitHub Pages

1. העלה את כל הקבצים לריפוזיטורי
2. הפעל GitHub Pages מהענף הרצוי
3. ודא ש-`manifest.json` קיים עם שדות `name`, `icons`, `start_url`

## בדיקה בטלפון

- כניסה ראשונה → מסך כניסה עם Google
- לאחר כניסה → דשבורד עם מספרי סיכום
- לחיצה על ➕ → אשף דיווח יומי (5 שלבים)
- לחיצה על אתר → פרטים + תמונות
- לחיצה על יומן → פרטים מלאים + כפתורי עריכה/מחיקה
- דוחות → בחר חודש → PDF או CSV
